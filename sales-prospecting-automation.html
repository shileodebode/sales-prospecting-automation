What is Sales Prospecting Automation?
An end-to-end automated prospecting pipeline built for Curacel's commercial team. It reads an Ideal Customer Profile from Google Sheets, searches Apollo.io for matching companies, cross-checks HubSpot to avoid duplicates, enriches leads with Precept market signals, finds decision-maker contacts, and uses GPT-4 to generate personalised LinkedIn, email, and WhatsApp messages — all ready to send. A Next.js dashboard lets the team monitor jobs, manage outreach, and track campaign replies in real time.

What Problem It Solves
⏱ Before This Agent
  • Sales reps manually searched for leads — hours per session
  • No consistent check against HubSpot — duplicate outreach common
  • Personalisation was generic or skipped entirely
  • No market signals used in messaging (funding, expansion, hiring)
  • Campaigns tracked in spreadsheets with no reply visibility
✓ After This Agent
  • Full ICP-to-message pipeline runs in minutes, not hours
  • HubSpot dedup built into every run — no duplicate outreach
  • GPT-4 personalises every message using company + contact context
  • Precept signals (funding, hiring, expansion) inform the copy
  • Gmail campaign sequencing with reply tracking and thread takeover
Core Capabilities
🔍 Company & Contact Discovery
Reads ICP filters from Google Sheets, searches Apollo.io for matching companies (up to 4× target for reliability), and sources decision-maker contacts via Apollo people search with enrichment fallbacks.
🧠 AI Message Generation
GPT-4 generates personalised outreach for LinkedIn, Email, and WhatsApp using company fit analysis, Precept market signals, and contact-level context from Tavily.
📬 Gmail Campaign Sequencing
Sends multi-step email sequences directly via Gmail. Tracks replies, auto-polls for new responses, maintains thread continuity, and allows reps to take over conversations when ready.

Architecture
⚙️

Two-service architecture: A Python FastAPI backend handles all pipeline orchestration and API integrations. A Next.js frontend provides the dashboard, job monitoring via SSE streaming, and the campaign management workspace. Both run as Docker containers deployed via GitHub Actions.

8-Step Pipeline
1
Load ICP Criteria
Read Ideal Customer Profile filters (industry, size, location, etc.) from Google Sheets.
2
Apollo Company Search up to 4× target, max 100
Search Apollo.io for companies matching ICP. Scans beyond the target count to ensure quality candidates after filtering.
3
HubSpot CRM Verification parallelised
Check each company against HubSpot — classify as net new, recycle, stale, or active. Skip existing contacts.
4
Deduplication
Skip companies already present in the output Google Sheet to prevent repeat processing.
5
Precept Intelligence
Enrich each company with Precept market signals — funding rounds, expansion plans, hiring activity — used as context for personalised messaging.
6
Load Outbound Scripts
Pull message templates and tone guidelines from Google Docs to frame GPT-4 generation.
7
Generate Messages parallelised
GPT-4 generates a fit analysis plus personalised LinkedIn, Email, and WhatsApp messages per contact. Tavily adds person-level web context.
8
Output Results
Write all prospects with generated messages and contact details to the output Google Sheet.

User Flows
Run a Prospecting Pipeline
  1. Define ICP criteria in Google Sheets
  2. Start a pipeline job from the dashboard
  3. Monitor 8-step progress via live SSE stream
  4. Review generated prospects + messages in output Sheet
  5. Initiate Gmail outreach from dashboard
Manage Campaign Replies
  1. System auto-polls for new email replies
  2. Replies surface in the workspace inbox
  3. Rep reviews thread continuity in the overlay
  4. Rep takes over the thread for manual follow-up
  5. Filter and scope views by saved workspace filters
